## salary-prediction-ml

Salary Prediction using Machine Learning Techniques

This project explores salary prediction using real-world data science salary data.

The goal is to build a complete machine learning workflow, starting with Simple Linear Regression and gradually extending the project with more advanced regression models and feature engineering techniques.

## Project Goals

- Perform Exploratory Data Analysis (EDA)
- Clean and preprocess the data
- Encode categorical features
- Train a Multiple Linear Regression model
- Evaluate model performance
- Visualize predictions and residuals
- Improve the project with more advanced models over time
